# Política de Seguridad

Este repositorio no debe contener secretos, credenciales, prompts privados, volcados de ejecución, registros, respaldos reales ni configuración específica de máquina.

Antes de publicar, ejecuta:

```bash
cd geo-control-plane
python scripts/validate_yaml.py
```

Usa el escaneo de secretos y la protección de_push de GitHub para repositorios públicos.
