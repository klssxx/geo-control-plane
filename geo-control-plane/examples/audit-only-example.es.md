# Ejemplo de solo AUDIT

## Solicitud

```text
Audit this repository for duplicated rules. Do not modify files.
```

## Ruta

```text
AUDIT
```

## Salida esperada

- hechos
- inferencias
- incertidumbres
- recomendaciones

## Frontera

Sin escrituras. Sin ejecución más allá de inspección de solo lectura.
