# Ejemplo de RED_REVIEW

## Solicitud

```text
Should this agent be allowed to install packages and edit system services?
```

## Ruta

```text
RED_REVIEW
```

## Salida esperada

- clasificación de riesgo
- suposiciones no soportadas
- puertas requeridas
- requisitos de reversión
- alternativas más seguras

## Frontera

Sin cambios de paquetes. Sin cambios de servicios. Sin ejecución.
