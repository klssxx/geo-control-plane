# LÍMITES DE EJECUCIÓN

Esta edición pública no es un controlador de ejecución.

## No incluido

- sin gateway
- sin trabajadores
- sin servidor MCP
- sin runtime Docker
- sin instalador de paquetes
- sin almacén de credenciales
- sin concesión de acceso al sistema de archivos
- sin perfiles privados

## Ejemplo de respaldo público

Usa solo referencias de respaldo anonimizadas:

```text
Backup created before write operation:
14_BACKUPS/example_backup_YYYYMMDD_HHMMSS/
```

No publiques contenido de respaldos reales.
